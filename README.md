# Random with custom distribution

I'm still working on it.
For now it returns a shuffled list of numbers within a specific range and distribution formula.

## Demo

min_value = -3
max_value = 3
step = 1
formula = lambda x:12-(x**2)
seeds = 500
round_to_step = True
