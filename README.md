# Random with custom distribution

I'm still working on it.<br>
For now it returns a shuffled list of numbers within a specific range and distribution formula.<br><br>

## Demo

min_value = -3<br>
max_value = 3<br>
step = 0.5<br>
formula = lambda x:12-(x**2)<br>
seeds = 500<br>
